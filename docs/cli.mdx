### Run a model

```
ollama run gemma3
```

#### Multiline input

For multiline input, press <span className="inline-flex items-center gap-1">
  <kbd className="bg-gray-100 dark:bg-gray-800 px-2 py-1 rounded border text-sm font-mono font-medium">Shift</kbd>
  <span className="text-sm">+</span>
  <kbd className="bg-gray-100 dark:bg-gray-800 px-2 py-1 rounded border text-sm font-mono font-medium">Enter</kbd>
</span>.

```
>>> """Hello,
... world!
... """
I'm a basic program that prints the famous "Hello, world!" message to the console.
```

#### Multimodal models

```
ollama run gemma3 "What's in this image? /Users/jmorgan/Desktop/smile.png"
```

### Generate embeddings

```
ollama run embeddinggemma "Hello world"
```

Output is a JSON array:

```
echo "Hello world" | ollama run nomic-embed-text
```

### Download a model

```
ollama pull gemma3
```

### Remove a model

```
ollama rm gemma3
```

### List models

```
ollama ls
```

### Sign in to Ollama

```
ollama signin
```

### Sign out of Ollama

```
ollama signout
```

### Create a customized model

First, create a `Modelfile`

```
FROM gemma3
SYSTEM """You are a happy cat."""
```

Then run `ollama create`:

```
ollama create -f Modelfile
```

### List running models

```
ollama ps
```

### Stop a running model

```
ollama stop gemma3
```

### Start Ollama

```
ollama serve
```

To view a list of environment variables that can be set run `ollama serve --help`
